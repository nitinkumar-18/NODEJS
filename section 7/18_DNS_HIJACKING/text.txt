Terminal me hosts file open karo:

sudo nano /etc/hosts

File ke end me add karo:

127.0.0.1   facebook.com
127.0.0.1   www.facebook.com

Save & exit:

CTRL + O → Enter
CTRL + X
🔄 DNS cache flush karo
sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder
