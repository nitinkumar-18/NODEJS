CIDR notation 
CIDR Notation (Classless Inter-Domain Routing) IP address ko compact aur efficient format mein likhne ka tarika hai.

Yeh batata hai:

✅ Network ka size kitna hai
✅ Kitne devices us network mein aa sakte hain


















🔹 👉 /24 → last number change hota hai

Example:

192.168.1.0/24

✔ Fixed part → 192.168.1
✔ Change hone wala → last number

Range:

192.168.1.0
to
192.168.1.255

👉 Devices: 254 usable

🔹 👉 /16 → last 2 numbers change hote hain

Example:

172.16.0.0/16

✔ Fixed → 172.16
✔ Change → last 2 numbers

Range:

172.16.0.0
to
172.16.255.255

👉 Devices ≈ 65,534

🔹 👉 /8 → last 3 numbers change hote hain

Example:

10.0.0.0/8

✔ Fixed → 10
✔ Change → last 3 numbers

Range:

10.0.0.0
to
10.255.255.255

👉 Devices ≈ 16 million 😲

🔹 Visual samajho

IP format:

A.B.C.D
CIDR	Fixed	Change
/8	A	B.C.D
/16	A.B	C.D
/24	A.B.C	D
🔹 Ek aur example
192.168.10.0/24

✔ Range → 192.168.10.0 – 192.168.10.255

192.168.0.0/16

✔ Range → 192.168.0.0 – 192.168.255.255
