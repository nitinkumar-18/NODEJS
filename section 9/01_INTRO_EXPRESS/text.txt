Example:

3️⃣ Route kya hota hai

Route = server ka rule

Matlab:

agar request ka path ye ho
to ye function run karo

Example:

app.get("/login",(req,res)=>{
   res.send("login page");
});

Yaha route hai:

/login
4️⃣ Difference samjho

Example request:

http://localhost:2700/login?user=tanish
Browser
full URL
Server ko milta hai
req.url = /login?user=tanish
Express route match karta hai
/login
5️⃣ Simple Table
Concept	Meaning
URL	full address
req.url	path + query received by server
route	server ka matching rule
